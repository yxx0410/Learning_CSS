## relative value
em: inherited parent's font size, 1em = font size
rem: inherited root font size, 1rem = root font size html size
vw,vh: 1vh = 1/100 view port height 
vmin,vmax : 1/100 view port min size or max size, screen can be rotate
0: remove a property value
numeric factor: some property like line-height
%: percentage of parent element property, default 100%
